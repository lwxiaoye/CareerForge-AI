# 联网搜索

## 适用场景

当学生提出需要联网获取最新信息的需求时使用，例如：
- 搜索公司背景、行业动态
- 读取招聘网站的岗位详情
- 抓取学生提供的 URL 链接内容
- 查询就业政策、薪资水平等实时信息

## 可用工具

本 Skill 提供两个内置工具：

### read_webpage（读取网页）

读取指定 URL 的网页内容，返回 Markdown 格式的正文。

参数：
- `url`（必填）：要读取的网页 URL
- `max_length`（可选）：返回内容最大字符数，默认 5000

适用场景：学生发来一个链接，需要读取其中的内容。

### web_search（联网搜索）

通过搜索引擎搜索关键词，返回搜索结果摘要。无需 API Key 即可使用。

参数：
- `query`（必填）：搜索关键词
- `num_results`（可选）：返回结果数量，默认 5

适用场景：需要搜索公司信息、行业动态、岗位详情等。

> 注：如果搜索无法返回结果，工具会回退并提示引导学生提供具体网址。

## 工作步骤

1. 判断学生需求是"给定 URL 读取"还是"关键词搜索"。
2. 如果是 URL，调用 `read_webpage` 读取内容。
3. 如果是关键词，调用 `web_search` 搜索信息。
4. 基于获取的内容，综合回答学生的问题。
5. 引用信息来源时附上 URL 链接。

## 注意事项

- 只能读取公开可访问的网页，需要登录的页面无法抓取。
- 抓取的内容可能包含噪声（导航栏、广告等），需要提取有效信息。
- 搜索结果仅供参考，应提醒学生交叉验证重要信息。
- 如果抓取失败，如实告知学生并建议其他途径。
