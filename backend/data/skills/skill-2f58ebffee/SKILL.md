# 岗位匹配分析

## 适用场景

当学生上传简历，并希望判断自己是否适合某个岗位时，使用这个 Skill。

## 输入

- 学生简历内容
- 目标岗位 JD
- 学生补充说明，可选

## 工作步骤

1. 提取岗位 JD 中的硬性要求、加分项、业务场景和隐含能力。
2. 从学生简历中提取教育背景、项目经历、技能关键词、成果证据。
3. 对照 JD 与简历，判断匹配项、缺失项和表达不清项。
4. 给出 0-100 的匹配度评分，并解释评分依据。
5. 输出可执行的简历优化建议和短期补强计划。

- 综合分：
- 结论：

- 优势 2：
- 优势 3：

### 差距项

- 差距 1：
- 差距 2：
- 差距 3：

### 简历优化建议

- 建议 1：
- 建议 2：
- 建议 3：

### 7 天补强计划

- 第 1-2 天：
- 第 3-5 天：
- 第 6-7 天：

## 注意事项

- 不要只堆关键词，要解释为什么匹配或不匹配。
- 如果简历证据不足，要明确指出“缺少证据”，不要臆造经历。
- 建议必须具体到学生可以修改的表达或可以补做的动作。
